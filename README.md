# Usage
## Installation
```
$ sudo npm install -g hexo-cli
```
## Run localhost
```
$ hexo s
```
Will serve
```
http://localhost:4000
```

## Create a new draft
```
$ hexo new draft <draft-name>
```
## Publish draft
```
$ hexo publish post <draft-name>
```
## Create Page
```
$ hexo new page <page-name>
```

## Publish draft
```
$ hexo publish <draft-name>
```

## Deploy
```
$ ./push-and-deploy.sh "<My comment>"
```
